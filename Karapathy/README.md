# Karpathy Learning Track

This folder contains notebooks and code inspired by Andrej Karpathy's lectures/courses and related GPT learning exercises.

## Sections

- [`GPT_from_scratch/`](./GPT_from_scratch/): tokenizer, bigram/GPT-style notebook experiments, BLT-related exploration
- [`gpt-2/`](./gpt-2/): GPT-2 training/eval experiments (`train_gpt2.py`, `hellaswag.py`, notebooks)
- [`makemore/`](./makemore/): Makemore course exercises / notebooks

## Portfolio-Relevant Highlights

- [`GPT_from_scratch/token.ipynb`](./GPT_from_scratch/token.ipynb): tokenization / minBPE-style learning implementation
- [`GPT_from_scratch/BLT.ipynb`](./GPT_from_scratch/BLT.ipynb): Byte Latent Transformer-inspired notebook experiment
- [`gpt-2/train_gpt2.py`](./gpt-2/train_gpt2.py): training-focused experiment code

## Why This Folder Exists

- Learn LLM internals by implementing pieces directly
- Build stronger intuition for tokenization, training loops, and model behavior
- Complement higher-level app/agent work in `random_code/`
